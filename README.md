# odin-recipes

This project is part of The Odin Project Foundations course.

The goal of this project is to build HTML chops. By completing this project, I will demonstrate my 
understanding of:
- Creating HTML pages
- Using heading, paragraphs, Links and Images
- Organizing files and directories
- Using Git and Github for version control

At the end of this project, I will have created a simple multi-page website and gained more 
confidence in writing semantic HTML and managing a project with Git.
